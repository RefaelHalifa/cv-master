# CV Design Specification

The approved visual style for all CVs generated by this plugin.
Apply to every user unless they explicitly request a different design.

## Color Palette

| Role | Hex | Description |
|---|---|---|
| Name | `#1A1A2E` | Dark navy blue |
| Section headers | `#2E6DA4` | Medium corporate blue |
| Body text / bullets | `#3A3A3A` | Dark gray |
| Job meta / subtitles | `#555555` | Mid gray |
| Secondary info | `#888888` | Light gray |
| Horizontal rules | `#CCCCCC` | Light gray |
| Links | `#2E6DA4` | Same blue as headers |

## Typography

| Element | Font | Size | Style |
|---|---|---|---|
| Name | Helvetica-Bold | 22pt | Centered |
| Section headers | Helvetica-Bold | 9pt | UPPERCASE |
| Body / bullets | Helvetica | 8.5pt | Regular |
| Job meta | Helvetica-Oblique | 8.2pt | Italic |
| Summary text | Helvetica | 8.8pt | Regular |

## Page Layout

- Page size: A4
- Margins: 16mm left/right, 14mm top, 12mm bottom
- Skills section: two-column table (label bold, value gray)
- Education section: two-column table (degree bold, institution gray)
- Horizontal rule: 0.6pt, `#CCCCCC`, under every section header
- ONE page only — non-negotiable

## Generation Method

Build with Python + ReportLab (Platypus). Generation script pattern:

```python
from reportlab.lib.pagesizes import A4
from reportlab.lib.units import mm
from reportlab.platypus import SimpleDocTemplate, Paragraph, Spacer, Table, HRFlowable
from reportlab.lib.styles import ParagraphStyle
from reportlab.lib import colors

PAGE_W, PAGE_H = A4
MARGIN_LR = 16 * mm
MARGIN_TOP = 14 * mm
MARGIN_BOT = 12 * mm

# Colors
NAVY    = colors.HexColor('#1A1A2E')
BLUE    = colors.HexColor('#2E6DA4')
DARK    = colors.HexColor('#3A3A3A')
MID     = colors.HexColor('#555555')
LIGHT   = colors.HexColor('#888888')
RULE    = colors.HexColor('#CCCCCC')

# Styles
name_style    = ParagraphStyle('name',    fontName='Helvetica-Bold',    fontSize=22, textColor=NAVY,  alignment=1)
header_style  = ParagraphStyle('header',  fontName='Helvetica-Bold',    fontSize=9,  textColor=BLUE,  spaceAfter=1)
body_style    = ParagraphStyle('body',    fontName='Helvetica',         fontSize=8.5,textColor=DARK,  spaceAfter=2)
meta_style    = ParagraphStyle('meta',    fontName='Helvetica-Oblique', fontSize=8.2,textColor=MID)
summary_style = ParagraphStyle('summary', fontName='Helvetica',         fontSize=8.8,textColor=DARK,  spaceAfter=3)

def section_header(title):
    return [Paragraph(title, header_style), HRFlowable(width='100%', thickness=0.6, color=RULE)]
```

## ATS Rules

- No tables used for layout (only for skills/education sections)
- No text boxes, columns, or graphics
- Standard section headings only
- Single-column main layout
- Keywords woven naturally into text — never stuffed
